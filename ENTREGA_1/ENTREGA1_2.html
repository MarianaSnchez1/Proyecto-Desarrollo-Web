Objetivos

Generales

  • Generar un espacio cómodo para el estudio
  • Mantener la información del usuario
  • Crear cada ambiente mencionado anteriormente

Específicos

  • Guardar la información personal del usuario como: nombre, apellido, horario
  • Hacer que el usuario pueda guardar sus libros, apuntes, y playlist
  • Crear una interfaz cómoda y sencilla (donde el usuario pueda entender desde que lo visualiza.
  • Organizar la información que se necesita (apuntes, libros, etc) de manera pertinente
  • Crear avisos para las tareas dependiendo de la fecha para entregar y la intensidad que se requiere.
