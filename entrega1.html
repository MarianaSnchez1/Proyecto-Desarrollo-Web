Idea Principal

Diseñar una página Web enfocada al estudio, creando varios ambientes enfocados al órden y estudio.

Los componentes que va a tener la página son los siguientes:

Sesión del usuario
Casa usuario que utilice la página va a poder entrar con su propia sesión, y así, poder guardar con su información personal (ya sea su horario, apuntes, entre otros).
Calendario
Este espacio cuenta con dos componentes principales: el horario del usuario y fechas de entrega. Así el usuario puede ir monitoreando sus metas de estudio, y organizar de manera eficiente su tiempo dedicado al estudio.
Apuntes
Va a haber un campo solo para que se puedan guardar los apuntes virtuales, para que sea más eficiente organizarlos y dividirlo dependiendo de las materias
Playlist
Para que sea más personalizado, el usuario va a poder agregar una playlist, y así poderla escuchar cuando esté trabajando
Libros
Igualmente, se podrá agregar libros, o documentos para usarlos como recursos.
Tareas
Las tareas que esté haciendo el usuario, se van a poder guardar y actualizar mientras las va haciendo, con el fin hacerle seguimiento y prevenir la pérdida de la información.

Objetivos

Generales

  • Generar un espacio cómodo para el estudio
  • Mantener la información del usuario
  • Crear cada ambiente mencionado anteriormente

Específicos

  • Guardar la información personal del usuario como: nombre, apellido, horario
  • Hacer que el usuario pueda guardar sus libros, apuntes, y playlist
  • Crear una interfaz cómoda y sencilla (donde el usuario pueda entender desde que lo visualiza.
  • Organizar la información que se necesita (apuntes, libros, etc) de manera pertinente
  • Crear avisos para las tareas dependiendo de la fecha para entregar y la intensidad que se requiere.
